## 1. Twelve Data API

### 1.1 개요

| 항목 | 내용 |
|------|------|
| **제공사** | Twelve Data Inc. |
| **커버리지** | 50개국 이상, 100만+ 금융 상품 |
| **무료 플랜** | 800 API 호출/일, 8 API 호출/분 |

### 1.2 제공 데이터 종류

| 카테고리 | 커버리지 | 갱신 주기 |
|----------|----------|-----------|
| **주식** | 미국 전체 + 90개 글로벌 거래소 | 실시간 (15분 지연) |
| **외환** | 140개 통화 | 실시간 |
| **암호화폐** | 180개 거래소, 10,000+ 페어 | 실시간 |
| **지수** | 글로벌 주요 지수 | 실시간 (일부 유료) |
| **기술지표** | 100개 이상 (SMA, RSI, MACD 등) | 시계열과 동일 |

### 1.3 주요 심볼

| 카테고리 | 심볼 | 설명 | 무료 지원 |
|----------|------|------|-----------|
| 환율 | USD/KRW | 달러/원 | ✅ |
| 환율 | EUR/USD | 유로/달러 | ✅ |
| 환율 | USD/JPY | 달러/엔 | ✅ |
| 지수 | SPX | S&P 500 | ⚠️ 제한 |
| 지수 | IXIC | NASDAQ | ⚠️ 제한 |
| 지수 | DJI | Dow Jones | ❌ 유료 |
| 암호화폐 | BTC/USD | 비트코인 | ✅ |
| 암호화폐 | ETH/USD | 이더리움 | ✅ |

## 2. 한국은행 ECOS API

### 2.1 개요

| 항목 | 내용 |
|------|------|
| **제공사** | 한국은행 경제통계시스템 |
| **커버리지** | 한국 경제 전반 (통화, 금융, 물가, 국민계정 등) |
| **무료 플랜** | 무제한 (과도한 호출 시 ERROR-602) |
| **API URL** | `https://ecos.bok.or.kr/api/` |

### 2.2 주요 서비스

| 서비스 | 설명 |
|--------|------|
| **StatisticSearch** | 통계 시계열 조회 (핵심) |
| **KeyStatisticList** | 100대 핵심 지표 |
| **StatisticTableList** | 통계 테이블 목록 |
| **StatisticMeta** | 통계 메타정보 |

### 2.3 주요 통계코드

#### 💰 금리

| 통계코드 | 항목코드 | 지표명 | 갱신 주기 |

| 722Y001 | 0101000 | 기준금리 | 금통위 (월 1회) |

## 3. API 비교 요약

| 항목 | Twelve Data | ECOS |
|------|-------------|------|
| **주요 용도** | 글로벌 금융시장 | 한국 경제 지표 |
| **실시간성** | 높음 (초~분) | 낮음 (일~월) |

| **환율** | 글로벌 페어 | 원화 기준 |

| **물가/GDP** | ❌ | ✅ |
| **기술 지표** | ✅ 100개+ | ❌ |

## 4. 활용 전략

### 4.1 데이터 소스 분류

| 구분 | API | 지표 예시 |
|------|-----|-----------|
| 실시간 시장 | Twelve Data | USD/KRW, BTC/USD, 주가지수 |
| 한국 금리 | ECOS | 기준금리, 국고채, CD |
| 한국 물가/GDP | ECOS | CPI, GDP 성장률 |

### 4.2 권장 갱신 주기

| 지표 유형 | 권장 갱신 | 캐시 TTL |
|-----------|----------|----------|
| 실시간 환율 | 5분 | 5분 |
| 금리 (일간) | 1시간 | 6시간 |
| 물가 (월간) | 1일 | 24시간 |

### 4.3 추가 가능 지표 (우선순위)

| 순위 | 지표 | API | 효과 |
|------|------|-----|------|
| ⭐⭐⭐ | 기준금리 | ECOS | 정책 금리 모니터링 |
| ⭐⭐⭐ | CPI | ECOS | 인플레이션 추적 |
| ⭐⭐ | GDP 성장률 | ECOS | 경기 사이클 |
| ⭐⭐ | CSI | ECOS | 소비자 심리 |
